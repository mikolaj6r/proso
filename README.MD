## Compile
Run in terminal: g++ -std=c++17 -lstdc++ -Wall -Wextra 'main.cpp' -o 'build/main'
OR
if You have NPM, you can run:
1) npm install -> to install package
2) npm run build -> to compile files
3) npm run start -> to run ./build/main
